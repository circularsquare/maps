ANCESTRY DOT MAP — COMMAND REFERENCE
=====================================

---- ONE-TIME SETUP ----

Install Python deps:
    pip install -r requirements.txt

Build tippecanoe (WSL Ubuntu, one-time):
    sudo apt-get install -y build-essential libsqlite3-dev zlib1g-dev
    git clone https://github.com/felt/tippecanoe.git
    cd tippecanoe && make -j && sudo make install


---- DATA PIPELINE (run in order) ----

1. Fetch Census ACS data:
    python fetch_data.py --state 36        # single state (NY = 36)
    python fetch_data.py --all             # all 50 states

2. Download shapefiles:
    python download_shapefiles.py --state 36 --areawater
    python download_shapefiles.py --all --areawater

3. Scatter dots → GeoJSON (also regenerates legend.json automatically):
    python scatter_dots.py --state 36      # single state
    python scatter_dots.py --all           # all 50 states

    To update only colors/legend without re-scattering (e.g. after subgroup changes):
    python scatter_dots.py --write-colors       # rewrites ancestry_colors.csv + legend.json
    python scatter_dots.py --write-legend       # rewrites legend.json only (reads existing GeoJSON)
    python scatter_dots.py --update-populations # updates only population column in ancestry_colors.csv + legend.json

4. Convert GeoJSON → PMTiles (run inside WSL for speed):

    cp /mnt/c/Users/anita/projects/maps/ancestrydots/data/processed/dots_all_1per100.geojson ~/dots.geojson
    tippecanoe -o ~/dots.pmtiles --name ancestry-dots --layer dots \
      --minimum-zoom 5 --maximum-zoom 9 \
      --drop-rate 1.0 --drop-densest-as-needed --maximum-tile-bytes 1500000 --no-feature-limit --full-detail 12 --force ~/dots.geojson
    cp ~/dots.pmtiles /mnt/c/Users/anita/projects/maps/ancestrydots/data/processed/dots_all_1per100.pmtiles


---- LOCAL DEV SERVER ----

    npx serve . -p 8080
    open http://localhost:8080

    (must be run from the ancestrydots directory)
    (Python's http.server won't work — it doesn't support HTTP range requests)


---- DEPLOY TO WEBSITE ----

1. Upload PMTiles to Cloudflare R2:
    - Go to https://dash.cloudflare.com > R2 Object Storage > "anita-maps" bucket
    - Upload dots_all_1per100.pmtiles (overwrites the old one)
    - Public URL: https://pub-ae551368cea941f39101e13c84d60bde.r2.dev/dots_all_1per100.pmtiles

    (PMTiles is NOT in the git repo — GitHub Pages + Cloudflare CDN breaks
    range requests with gzip encoding. R2 handles them correctly.)

2. Copy legend + index to the website repo:

    cp ~/projects/maps/ancestrydots/index.html  ~/projects/website/ancestrydots/index.html

    cp ~/projects/maps/ancestrydots/data/processed/legend.json ~/projects/website/ancestrydots/data/processed/legend.json

3. Commit and push:

    cd ~/projects/website
    git add ancestrydots/
    git commit -m "update ancestry dot map"
    git push

Live at: https://circularsquare.github.io/ancestrydots/
